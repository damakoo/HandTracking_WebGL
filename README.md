# HandTracking_WebGL

UnityのWebGLでMediapipeのHandTrackingを使用したい人のためのサンプルプロジェクト

# DEMO

https://damakoo.github.io/HandTracking_WebGL/


# Features

Google Mediapipe を使用．JavaScript側でハンドトラッキングを行い，LocalStorage経由でUnityWebGLに結果を送信している．
